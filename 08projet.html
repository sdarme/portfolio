Édition - Protocole au lancé de dés

2019


Un protocole à jouer avec deux dés m’a été attribué. J’en ai défini les règles qui m’ont alors permis de définir chaque élément de cette édition : textes, papiers, fil, reliure, pagination, iconographie. Les yeux fermés, j’ai sélectionné les sources dans ma bibliothèque. Protocole donné: lancer les dés jusqu’à faire le même double, préalablement choisi, deux fois de suite. Appliquer une règle à chaque chiffre.

I received a protocol to play with two dices. I created the rules then it allowed me to define one rule for each element of this publishing like texts, paper, bending thread, style of thread, pagination or iconography. Eyes closed I selected all the items in my personal library. The protocol I was gaved said : Throw the dices until you make twice the same double you already choose. Give a rule to each number.
